# Security Policy

## Supported Versions

Only the latest release of each project is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in any ZenHelix project, please report it responsibly:

1. **Do NOT open a public issue**
2. Email: dm.medakin.online@gmail.com
3. Include: description, reproduction steps, affected versions, and potential impact

## Response

- Acknowledgment within 48 hours
- Assessment and fix timeline within 7 days
- Public disclosure after fix is released

## Scope

This policy applies to all public repositories in the [zenhelix](https://github.com/zenhelix) organization.
